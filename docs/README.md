## 구현 목록

### 구현 기능 목록

### 예외 처리 목록
- 입력값이 형식을 벗어나는 경우(입력받는 값은 전부 숫자임을 참고)
- 입력한 가격이 1000원 단위가 아닌 경우
- 입력한 가격이 1000원 미만인 경우
- 입력된 추리숫자가 6개가 아닌 경우
- 정답/추리숫자의 범위가 1~45를 벗어나는 경우
- 추리숫자가 중복되는 경우
- 정답숫자 + 보너스숫자가 중복되는 경우
